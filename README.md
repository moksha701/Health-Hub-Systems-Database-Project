# HealthHub Systems

📌 **Purpose of the Project**

HealthHub Systems is a database system designed to enhance the operational efficiency of health insurance administration. The main objectives are:

* To manage subscriber enrollment, dependents, and insurance plan details.
* To streamline claims processing and track approval or denial outcomes.
* To enable healthcare insurance providers to access and analyze key business metrics.

---

🔍 **Analysis Performed**

1. Entity Relationship Diagram (ERD) Design
2. Relational Data Modeling (RDM)
3. Database Normalization (1NF, 2NF, 3NF, BCNF)
4. SQL Database Implementation (DDL & Insert Statements)
5. Business Scenario Queries

---

📌 **Results**

The system provides insights into:

* Subscriber enrollment counts by plan type.
* Claims generated per provider per month.
* Average billed amounts by plan type.
* Percentage of premiums paid out in claims.
* Most utilized provider facility types.
